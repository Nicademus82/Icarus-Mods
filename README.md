# Nics_Overhual

**Dropship Slots**

* Dropship loadout increased to 30 slots (Nics_Overhaul_30_P)
* Dropship loadout increased to 35 slots (Nics_Overhaul_35_P)

**Envirosuits**
* All suits have +1 slots than normal for upgrade modules
* Inarus Envirosuit has 4 module slots & includes bonus stats for stamina and stamina regen
* First Cohort suit has 4 module slots, added 20% heat/cold resistance & includes bonus stats for stamina and stamina regen

**MODULES AND BACKPACKS**
* Movement speed module increased to 10% 
* Carry weight module increased to 25kg 
* Survivors backpack increased movment speed to 10%
* Gathers backpack increased to 12 slots & -50% carry weight of ores & wood 
* Larkwell storage backpack increased to 24 slots & added 50kg weight capacity

DISCLAIMER: these mods are intended for Offline and/or solo play. It can potentially ruin the game for others when playing together. If you play with a regular group, please recommend all parties install the mod to prevent adverse effects. Running mods in a group setting without everyone else running them can cause all sorts of problems (items missing, crashes, etc.)

INSTALLATION: if you do not already have one, you need to create a new folder and name it 'mods' in the following directory - Disk > Program Files (x86) > steamapps > common > Icarus > Icarus > Content > Paks. Download .pak file, and drag and drop into the mods folder you created. Good to go!
